Fading Parkour 2 Endpoints

GET /roblox/fp2/leaderboard.json

Returns a list of the top 20 roblox ids with the most wins in fading parkour 2.

Get /roblox/fp2/rank.json

NOTICE: Custom ranks are deprecated, do NOT use this api for new work.

Returns a list of all the roblox ids with custom ranks.
